# Ashil Kumar — React Portfolio

Premium React developer portfolio built with React 19, Vite, TypeScript, SCSS, Zustand, and TanStack Query.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Stack

- React 19 + Vite + TypeScript
- SCSS architecture with design tokens
- Zustand (theme persistence)
- TanStack Query (contact form mutation)

## Structure

```
src/
├── components/
├── data/
├── hooks/
├── store/
├── styles/
├── App.tsx
└── main.tsx
```
